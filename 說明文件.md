# 專案說明文件

## 本地預覽方法

由於本專案採用 Vue.js 3 ES Modules (`type="module"`)，直接點擊 `index.html` 檔案可能因瀏覽器的安全性限制（CORS）而無法正常載入腳本。必須透過本地伺服器環境運行。

### 方法一：使用 Python (推薦，無需安裝)
如果您是 Mac 或 Linux 使用者，終端機通常已內建 Python：

1. 開啟終端機 (Terminal)。
2. 切換至專案目錄：
   ```bash
   cd /Users/jing/Downloads/jing-record
   ```
3. 啟動伺服器：
   ```bash
   python3 -m http.server 8000
   ```
4. 預覽連結：[http://localhost:8000](http://localhost:8000)

### 方法二：使用 Node.js (http-server)
如果您已安裝 Node.js：

1. 在專案目錄執行：
   ```bash
   npx http-server -p 8000
   ```
2. 預覽連結：[http://localhost:8000](http://localhost:8000)

## 元件說明

### Header 管理
專案的最上方標題與動作按鈕由 `js/components/app-header.js` 統一管理。

- **標題**：`NICHI-NICHI` 字樣。
- **視覺穩定性**：統一頁首高度為 `h-16` (64px)，確保切換頁面時標題位置完全固定不跳動。
- **右側動作區**：
    - 在 **Overview** 與 **Stats** 頁面：顯示「幣別切換」組件，全站數據同步。
    - 在 **History** 頁面：顯示「CLEAR 篩選清除」按鈕。
- **管理方式**：使用 Vue 的 `v-if` 與 `currentTab` 進行條件渲染。

### 通用組件 (UI Components)
- **自訂選單 (`app-select.js`)**：取代原生 `<select>`。
    - **特點**：支援自訂圓角與邊距，具備慣性彈跳動畫。
    - **視覺**：採用毛玻璃過濾器與半透明背景，適配深色/淺色模式。

### 統計 (Stats) 管理
- **時間篩選**：支援「按月」、「自訂區間」與「所有數據」。
- **數據模式**：可切換「個人份額」與「排除專案支出」核取方塊。
- **圖表分析**：使用 Chart.js 進行類別佔比與支付方式分析，數據隨幣別即時換算。
