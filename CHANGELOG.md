# 更新日誌 - 2026/02/07

## [1.3.1] - 2026-02-07
### Added
- **自訂下拉選單組件 (AppSelect)**：開發全新的全域選單組件，具備毛玻璃效果 (backdrop-blur) 與更加細膩的彈性動畫 (12px spring-up)。
- **歷史頁面篩選優化**：搜尋列中的「類型篩選」已升級為自訂下拉選單，與全站視覺統一。

### Changed
- **動畫系統微調**：全站 `spring-up` 動畫位移從 20px 減至 12px，提供更輕盈、穩定的視覺回饋。
- **統計頁面重構**：專案選單已替換為 `AppSelect`，並優化了篩選核取方塊的佈局。

## [1.3.0] - 2026-02-07
### Added
- **統計頁面「所有數據」篩選**：Stats 頁面新增「所有」時間範圍，自動計算全歷史區間的總額與日均花費。
- **全局幣別切換連動**：Stats 頁面現在與 Overview 頁面共享頁首的幣別切換按鈕，達成全站數據幣別同步。

### Changed
- **頁首高度標準化**：統一所有分頁的頁首高度 (h-16)，解決切換頁面時標題跳動問題。
- **統計頁面過濾器重構**：
    - 將「總和/個人」切換按鈕改為更精簡的「僅顯示個人份額」核取方塊。
    - 將過濾選項改為左右並列佈局，提升畫面空間利用率。
    - 統一所有日期選擇輸入框的高度與樣式。

## [1.2.0] - 2026-02-06
### Added
- **頁首幣別切換**：幣別切換按鈕移動至頁首「Nichi-Nichi」右側，樣式更簡約且全局同步。
- **時區鎖定 (Timezone Locking)**：編輯紀錄時會自動鎖定原始時區，防止跨時區編輯導致時間偏移。
- **標準化時間格式**：全站統一使用 `YYYY/MM/DD HH:mm UTC+X` 字串存儲與顯示。

### Fixed
- **時區偏移修復**：移除後端強制的 GMT+9 格式化，達成「所見即所得」。
- **幣別符號錯誤**：修正歷史明細與專案統計中硬編碼的 `¥` 符號課題。
- **遷移腳本索引 Bug**：修復 `migrateLegacyData` 對於台幣資料原始金額映射錯誤的問題。
- **數據同步強化**：確保統計頁面 (Stats) 與專案頁面 (Project Detail) 均能正確響應全局幣別切換。

### Changed
- **歷史分組優化**：月份標題與日期分隔符號統一為 `/`，提升視覺穩定性與排序準確度。
